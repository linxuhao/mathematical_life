## 0. The two axes

### Axis 1 — STATIC vs DYNAMIC
- **STATIC**: one snapshot per model; compare across models / prompt-modes (R vs H vs control).
  Question: *where do cognitive modes sit, and how do they differ structurally?*
- **DYNAMIC**: a checkpoint series of one model across training; compare across steps.
  Question: *how does training reshape structure, and does structure track capability?*

### Axis 2 — the 5 point-cloud channels (canonical, FINDINGS §3)
A cloud's information splits by which geometric transform preserves it. The user's
shorthand "coordinate / shape / angles / …" maps onto these:

| # | Channel | "Plain" name | What it represents |
|---|---|---|---|
| 1 | **Position** (centroid, union, bottleneck) | coordinate | which region of activation space a mode occupies |
| 2 | **Scale** (spread / magnitude) | size | overall spread; *normalized out* by FROZEN_V5 (was the tokenwise length confound) |
| 3 | **Orientation / effective dim** (participation ratio, EVR1) | shape / angles | dimensionality & anisotropy of the mode |
| 4 | **Connectivity** (homology) | holes | 4a β₀ clusters · 4b β₁ loops · 4c β₂ voids |
| 5 | **Persistence** (survival_rate, PHI) | robustness | how robustly a shape survives as the scale ε grows |

### Constants shared by ALL measurements — FROZEN_V5 protocol
- Layer = **L/2** (middle layer), last real token (mask-based, left-pad — NOT the `hidden[:,-1]` pad bug).
- Filtration ε = **0.03 × ε_max** for the headline β₁ ("filtered"); β₁_raw = all loops, no lifetime cut.
- Static clouds = **1319 prompts**; dynamic eval clouds = **N=300/domain**.
- dtype **bf16** (fp16 → NaN on ROCm for small models). Engine = `actopo` (reproduces betti_v3 186/186).

## STATIC EXPERIMENTS

### S0 — Baselines (Experiment 0)
- **Variables**: input type {real activations, Gaussian noise, INT4-quantized, fp32}; layer index.
- **Constants**: FROZEN_V5, one model family.
- **Measured**: β₁ (raw+filtered); reproducibility (bit-identity).
- **Matrix**: STATIC × ch4b (β₁) + ch5 (persistence sanity).
- **Result**: Gaussian → β₁_filtered=0 (raw loops exist, none survive); INT4 preserves 92%; L/2 is the unique informative layer; extraction bit-identical (deterministic). *Establishes the signal is real, not noise.*

### S1 — Static topology across models (Part A)
- **Variables**: model {23 models, base vs instruct, 0.5–9B, Qwen2.5/3/3.5, Gemma}; prompt-mode {8 types: Reasoning, Hallucination, Control, …}.
- **Constants**: FROZEN_V5, 1319 prompts.
- **Measured**: β₁ (raw+filtered) per (model, mode).
- **Matrix**: STATIC × ch4b (β₁).
- **Result**: β₁ is the **unique** R-vs-H discriminator (p=0.005). **β₁ ≠ capability** (instruct β₁_R = 40→80→90→1 across 0.8/2/4/9B; Spearman(size,β₁)=−0.20, non-monotone). → refutes Prediction 1 statically. *Verified.*

### S2 — Mode position / separation (Part B)
- **Variables**: mode pair {R, H, Control, …}; metric {centroid distance, bottleneck, union}.
- **Constants**: FROZEN_V5; within-basis only (Iron Law: never cross base/instruct).
- **Measured**: do two modes occupy separable regions? (648/656 SEPARATE).
- **Matrix**: STATIC × ch1 (Position).
- **Result**: modes are topologically distinguishable (648/656); bottleneck R↔H recomputed exact (1.804/1.665). *This is the most robust positive — the "weak form" that survives.*

### S3 — Persistence / PHI (Part C)
- **Variables**: model (14 with full curves); base vs instruct.
- **Constants**: FROZEN_V5.
- **Measured**: PHI = survival-rate ratio; persistence curves.
- **Matrix**: STATIC × ch5 (Persistence).
- **Result**: 9B-Inst PHI crosses 1.0 (collapse direction). **Number corrected**: canonical PHI = **1.5→0.65** (survival-rate @1%), NOT the old 2.27→0.31 (survival-count @3%). Cite 1.5→0.65. *Weakest gauge; solid only at 9B.*

### S4 — Tripartite test (Part F)
- **Variables**: answer quality {R_true n=204, R_flawed n=1115, H n=1319}.
- **Constants**: 2B-Inst, FROZEN_V5, zero-shot.
- **Measured**: β₁ across the three sub-modes.
- **Matrix**: STATIC × ch4b.
- **Result**: graded structure across true/flawed/hallucinated. (Supportive of mode-structure, not of β₁=capability.)

### S5 — Channel inventory (phaseC, β₀/β₁/β₂ scale-scans, PR)
- **Variables**: channel {β₀, β₁, β₂, participation-ratio}; filtration scale ε {1%–30% ε_max}; mode {R, H, Control}.
- **Constants**: FROZEN_V5 clouds.
- **Measured**: which channel separates which modes, at which scale.
- **Matrix**: STATIC × ch3 (PR) + ch4a (β₀) + ch4b (β₁) + ch4c (β₂) + ch5 (scale).
- **Result**:
  - **ch3 PR**: separates Control (~18-dim) from R/H (~37); convergent 9B collapse (4B 81→9B 37); does NOT separate R vs H.
  - **ch4a β₀**: separates R/H only at **coarse** scale (20–30% ε_max, p<0.001 — H coalesces faster); dead at 3%.
  - **ch4b β₁**: separates R/H at **fine** scale (1–5%, best ~2%); β₀ & β₁ are **complementary**.
  - **ch4c β₂**: MEASURED = EMPTY (raw 30–50 loops but all ephemeral, ~0 survive 3%). Topology confined to dims 0–1.
  - Convergent 9B-Inst collapse across 3 independent channels (β₁ 90→1, PHI >1→<1, PR 81→37) → not a β₁ artifact.

### S6 — Geometry-predicts-correctness (C3) + controls
- **Variables**: predictor {surprisal, prompt-end activation (pre-reasoning), decision-token activation (post-reasoning), surface-difficulty scalars}; control {none, question-length stratification, within-question centering}.
- **Constants**: 2B model, L/2, GSM8K items, 5-fold CV ROC-AUC.
- **Measured**: AUC for predicting per-item answer correctness.
- **Matrix**: STATIC × ch1 (Position, as a per-item geometric coordinate).
- **Result (3-tier, fully controlled)**:
  1. surface difficulty (q-length) = 0.649 — most of the signal, a "long → wrong" prior.
  2. **pre-reasoning comprehension** (length-stratified prompt-act) = **0.622** — genuine, clean: the model anticipates solvability at read-time beyond length.
  3. within-question reasoning-process (confound-free same-Q centering) = **0.584** — weak but non-zero.
  - Raw decision-token AUC 0.747 *looks* strong but is **mostly difficulty**, not reasoning. Confident-error: when wrong, gold answer +4.70 nats more surprising (model confidently wrong).
  - Defensible claim = tier 2 (comprehension), NOT "geometry tracks reasoning-correctness."

### S7 — Villain prefill probe
- **Variables**: forced reasoning {correct, wrong} on the same prompt; prefill only (no decode).
- **Constants**: same prompt pairs, decision-token activation + surprisal.
- **Measured**: does the internal state distinguish correct vs forced-wrong reasoning?
- **Matrix**: STATIC × ch1.
- **Result**: prototype-level; folded into the C3 confident-error finding.

## DYNAMIC EXPERIMENTS (training checkpoint series)

### D1 — 0.5B-Base SFT on GSM8K (30 ckpts)
- **Variables (IV)**: training step.
- **Constants**: Qwen2.5-0.5B-Base, GSM8K SFT, FROZEN_V5.
- **Measured (DV)**: GSM8K acc, β₁, PHI.
- **Matrix**: DYNAMIC × ch4b + ch5.
- **Result**: acc 22.5→32.5%; PHI FLAT. (Undertrained model: training builds structure.) NOTE: earlier "0.5B null" was a pad-bug artifact; corrected β₁_R=22.

### D2 — 2B-Instruct SFT (60 ckpts) + chat-template fix
- **Variables (IV)**: training step; **eval format {raw text, chat template}**.
- **Constants**: Qwen3.5-2B-Instruct, FROZEN_V5.
- **Measured (DV)**: acc, β₁, R−H gap, PHI.
- **Matrix**: DYNAMIC × ch1 (R−H gap) + ch4b + ch5.
- **Result**: **"SFT erodes reasoning topology" is FORMAT-DEPENDENT** — raw text r(step,gap)=−0.68; chat template r=+0.10 (flat). The erosion was a format artifact. *This is why the collapse experiment used a BASE model + format-robust eval.*

### D3 — 2B-Base SFT (actopo re-run)
- **Variables (IV)**: training step.
- **Constants**: Qwen3.5-2B-Base, GSM8K SFT (gentle, lr=1e-5), FROZEN_V5.
- **Measured (DV)**: acc, β₁, R−H gap.
- **Matrix**: DYNAMIC × ch1 + ch4b.
- **Result**: raw-text gap erodes (r=−0.81) while GSM8K rises — but see D2: format-dependent. These same 30 ckpts are reused as the **gentle arm** below.

### D4 — Tokenwise (Part E) + aha-moment + thermometer + TQA (supplementary)
- **Variables (IV)**: token position within a trajectory (tokenwise); reasoning gauge → OOD benchmark (thermometer); correct vs incorrect trajectory (aha).
- **Constants**: per-test; partial-correlation-on-step / length / problem-identity controls.
- **Measured (DV)**: β₁(token), gauge↔OOD acc, β₁ AUC for correctness.
- **Matrix**: DYNAMIC × ch4b.
- **Result**: ALL **DEAD** at power. Tokenwise null; aha = between-problem length confound (matched-k AUC ~0.5); thermometer partial|step ≈ 0 across models; TQA "collapse" = letter-output bias (real effect modest ~6pp). Killed by sound controls, not our bugs.

### D5 — COLLAPSE EXPERIMENT (the capstone): off-target forgetting
Shared design (both arms):
- **Variables (IV)**:
  - between-arm: **learning rate** {gentle 1e-5, aggressive 5e-5} — the "force of the global hammer."
  - within-arm: **training step** (checkpoint series).
  - **domain**: A = GSM8K (trained) vs B = {ARC-Easy, HellaSwag} (UNRELATED / off-target).
- **Constants**: **Qwen3.5-2B-Base** (same model both arms; base step0 reproduces EXACTLY across arms), "Q:/A:" GSM8K training format, fp32 train + grad-clip 1.0, FROZEN_V5 measurement, N=300/domain, **likelihood-MC** for B (format-robust, no generation — controls the D2 format confound), generative for A, L/2.
- **Measured (DV)**, per (step, domain): accuracy; β₁ raw; β₁ filtered; survival_rate; C3-comprehension AUC.
- **Matrix**: DYNAMIC × ch1 (C3) + ch4b (β₁) + ch5 (survival).
- **Crux statistic**: partial_corr(B-topology, B-acc | step) — distinguishes *load-bearing* (collapse drives forgetting) from *epiphenomenal* (both just decline with step).

**D5a — GENTLE arm** (reuse 30 D3 ckpts, NO training):
- **Q1**: A learns (0.086→0.130, r=+0.74); **B FLAT** (ARC 0.690→0.696; Hella 0.531→0.523). No forgetting.
- **Q2/Q3/Q5**: topology↔B-acc correlations weak and **flip sign across B-domains** → noise.
- Within trained A: β₁_raw ↑ (+0.62), survival ↓ (−0.65), C3 ↑ (+0.40) = learning reshapes structure.
- **Verdict**: gentle regime induces no forgetting → can't test the thesis. → ran aggressive.

**D5b — AGGRESSIVE arm** (retrain base, lr=5e-5, 3 ep, 29 ckpts; loss 1.27→0.22):
- **Q1 forgetting = YES** (base→end): A 0.157→**0.42** (+27pp); B_arc 0.693→**0.534** (−16pp); B_hella 0.530→**0.433** (−10pp). Front-loaded (B craters by ~step1600, ARC partly recovers, Hella stays down).
- **Q2 topology also falls**: B_hella β₁_raw r(step)=−0.93, survival −0.76; B_arc survival −0.62.
- **Q3 CRUX = BYSTANDER**: partial_corr(B-topo, B-acc | step) ≈ 0 everywhere (ARC β₁ +0.36 p=.05 marginal & UNREPLICATED; all others |r|<0.2; HellaSwag all ~0; signs flip across domains). Raw corrs also |r|<0.22. **Topology and forgetting decline in parallel as functions of step; collapse carries no step-independent signal about forgetting.**
- **Q5**: NO signal tracks forgetting, incl C3 (drifts UP while acc falls).

**D5c — CONFIDENT-FORGETTING (the positive longitudinal result)** `tools/aggressive_confidence.py`:
- **Variables**: step; domain B={ARC, HellaSwag}. **Measured**: confidence = max softmax over per-choice avg-logprobs (at the DECISION, after choices seen); confidence|correct vs |wrong; overconfidence (conf−acc); ECE. **Matrix**: DYNAMIC × ch1/ch5 (a calibration readout, not homology).
- As B-accuracy falls, **confidence RISES** (r(step)=+0.94, p<.001 both domains); **conf|wrong rises** (r≈+0.95); **overconfidence flips negative→positive** (ARC −0.08→+0.16; Hella −0.07→+0.13); **ECE nearly doubles** (ARC 0.11→0.17; Hella 0.07→0.13). corr(overconfidence, acc) ≈ −0.57.
- The model becomes **confidently wrong** as it forgets — the longitudinal version of the static confident-error finding (+4.7 nats, S6). conf-correct−conf-wrong gap shrinks 0.10→0.06 (confidence decouples from correctness).
- Mechanism caveat: GSM8K overfit sharpens output distribution globally (↓entropy → ↑max-prob), which inflates confidence everywhere — but that IS the miscalibration (peakedness unjustified by B-competence); robust because measured vs actual accuracy.
- **KEY CONTRAST**: topology (β₁/survival) = BYSTANDER (D5b Q3); **calibration = TRACKS forgetting**. The legible signature of catastrophic forgetting is overconfidence, not the Betti number.

**⚠ RAW vs FILTERED — the death claim (must double-check):**
On trained task A, capability rises 2.7× while:
- **β₁ RAW: 107 → 77** (r=−0.45) — a ~28% drift, **NOT a collapse**. ~80 loops remain. [verified vs aggressive_arm.json step0→11208]
- β₁ filtered (>3%): 4 → 0; survival_rate: 27 → 9 (r=−0.89).
- The "→0" is the FILTERED count only, caused by the survival drop pushing loops under the 3% lifetime cut — the same filter-fragility as the "9B collapse" (raw 946→1074).
- **Therefore**: the paper's raw-spirit "death" (structure vanishing) is **NOT** triggered — raw loops persist. The clean, filter-independent claim is the weaker one: **β₁ does not track capability** (holds for raw AND filtered). Do NOT state "β₁=0 = dead at peak" as if it were the paper's raw death.

## ONE-LINE VERDICTS (for the paper)
- **Survives (weak form)**: modes are topologically distinguishable (ch1 position, 648/656);
  learning reshapes structure (ch4b/ch5 dynamic). Comprehension is legible pre-reasoning (C3 tier 2).
- **Falsified (strong form)**: β₁ = reasoning/capability meter (S1, D5: β₁ ⊥ capability, raw & filtered);
  β₁-collapse = death (raw doesn't collapse; where filtered does, capability PEAKS); topology is
  load-bearing for capability (D5 crux: bystander, partial|step ≈ 0).
- **Mode separation (ch1) and β₁≠capability (ch4b) are the two filter-independent pillars.**
- **Positive longitudinal result**: catastrophic forgetting's legible signature is **calibration**
  — the model becomes confidently wrong (overconfidence flips +, ECE doubles) as it forgets —
  **not** topology (β₁/survival are bystanders). The "confidence/comprehension" axis the project
  kept circling back to is where the dynamic signal actually lives.
